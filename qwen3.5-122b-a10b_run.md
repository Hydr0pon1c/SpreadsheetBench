# Qwen3.5-122B Trace2Skill Random-Split Run

This run repeats the Trace2Skill reproduction on `spreadsheetbench_verified_400` with a reproducible random 200/200 split, using `qwen3.5-122b-a10b` for inference, skill0 generation, and combined error+success patch synthesis.

## 0. Current Split Status

The existing split generated by `trace2skill/prepare_splits.py` is **not random**. It uses dataset order:

- `data/splits/verified_evolve_200.json`: first 200 rows in `dataset.json`
- `data/splits/verified_test_200.json`: next 200 rows in `dataset.json`

The random split below writes new files and does not modify those fixed-order split files.

## 1. Create Random 200/200 Split

Run from the repository root. Change `SEED` only if you intentionally want a different split.

```bash
python3 - <<'PY'
import json
import random
from pathlib import Path
from datetime import datetime, timezone

SEED = 0
dataset_name = "spreadsheetbench_verified_400"
dataset_path = Path("data") / dataset_name / "dataset.json"
output_dir = Path("data/splits")
output_dir.mkdir(parents=True, exist_ok=True)

dataset = json.loads(dataset_path.read_text())
indices = list(range(len(dataset)))
random.Random(SEED).shuffle(indices)
evolve_indices = indices[:200]
test_indices = indices[200:400]

def build(name, chosen):
    records = []
    for idx in chosen:
        item = dataset[idx]
        records.append({
            "index": idx,
            "id": str(item["id"]),
            "instruction_type": item.get("instruction_type"),
            "spreadsheet_path": item.get("spreadsheet_path"),
        })
    return {
        "dataset": dataset_name,
        "split": name,
        "policy": f"random_seed_{SEED}_200_evolve_200_test",
        "created_at": datetime.now(timezone.utc).isoformat(),
        "seed": SEED,
        "count": len(records),
        "ids": [record["id"] for record in records],
        "records": records,
    }

(output_dir / f"verified_random_seed{SEED}_evolve_200.json").write_text(
    json.dumps(build("evolve", evolve_indices), indent=2, ensure_ascii=False) + "\n"
)
(output_dir / f"verified_random_seed{SEED}_test_200.json").write_text(
    json.dumps(build("test", test_indices), indent=2, ensure_ascii=False) + "\n"
)
print("wrote random split with seed", SEED)
PY
```

Expected files:

```text
data/splits/verified_random_seed0_evolve_200.json
data/splits/verified_random_seed0_test_200.json
```
